## 关于 Sublime Text 台湾

「Sublime Text 台湾」是由爱好者自行成立与维护的网站，目的在于分享各种 Sublime Text 文字编辑器的使用技巧，以及相关的新闻消息，推广好用的开发软体一起增进<del>宅指数</del>生产力！
